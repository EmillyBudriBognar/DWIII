# Desenvolvimento Web III (DWIII)

Repositório dedicado às atividades e aulas da disciplina de **Desenvolvimento Web III**, ministrada pelo Professor **Vinicius Heltai**.

## 📂 [01-Introducao-Node-04-03](./01-Introducao-Node-04-03)
**Introdução ao Node.js (Aula 04/03)**
- **Como Rodar:** `node src/abreArquivoJson.js` (ou qualquer outro arquivo dentro de `src/`).
- **Acesse:** `http://localhost:3000`.

### 📂 [02-Rotas-e-JSON-11-03](./02-Rotas-e-JSON-11-03)
**Rotas e Frontend (Aula 11/03)**
- **Como Rodar:** `node src/server.js`.
- **Acesse:** `http://localhost:3000` para ver o HTML/JS ou `http://localhost:3000/rota1` para o JSON direto.

### 📂 [03-Atividade-Design-Patterns](./03-Atividade-Design-Patterns)
**Atividade Design Patterns**
- **Como Rodar:** `node src/VeiculosDeTransporte.js` (ou outro dentro de `src/`).
- **Acesse:** `http://localhost:3000/terrestre` ou `/aereo`.

---

## 🚀 Tecnologias Utilizadas
- **Node.js**: Ambiente de execução JavaScript server-side.
- **Vanilla JavaScript**: Lógica do servidor e manipulação de rotas.
- **JSON**: Formato de intercâmbio de dados.
- **HTML/CSS**: Interface básica para testes de rotas.
